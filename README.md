# Loop Agent — ADOM-LOOP Operations Guide

**Loop** is an OpenClaw agent that helps you set up, run, and get the most out of [ADOM-LOOP](https://github.com/AutoBridgeSystems/ADOM-LOOP) — the AutoBridge agentic app builder.

Loop knows how to:
- Walk you through local ADOM-LOOP installation step by step
- Guide the spec interview so generated apps are agent-operable (not just UI-clickable)
- Monitor running builds and troubleshoot failures
- Manage generated apps via API instead of browser
- Maintain a growing knowledge base of learnings and gotchas

---

## Requirements

- [OpenClaw](https://openclaw.ai) installed and running
- An Anthropic API key configured in OpenClaw
- Access to the `AutoBridgeSystems/ADOM-LOOP` repo (private — request access from AutoBridge)

---

## Setup

### 1. Clone this repo

```bash
git clone git@github.com:LandonMMoore/loop-agent.git ~/loop-agent
```

### 2. Fill in your USER.md

Edit `~/loop-agent/USER.md` with your name and preferences. Loop reads this every session.

### 3. Register the agent in OpenClaw

Add the following to your `~/.openclaw/openclaw.json` under `agents.list`:

```json
{
  "id": "loop",
  "name": "Loop",
  "workspace": "/Users/YOUR_USERNAME/loop-agent",
  "agentDir": "/Users/YOUR_USERNAME/.openclaw/agents/loop/agent",
  "model": "anthropic/claude-sonnet-4-6"
}
```

Replace `YOUR_USERNAME` with your actual macOS username.

### 4. Restart OpenClaw

```bash
openclaw gateway restart
```

### 5. Invoke Loop

From your main OpenClaw session:
```
loop agent: help me set up ADOM-LOOP
```

Or via CLI:
```bash
openclaw agent --agent loop --message "help me set up ADOM-LOOP"
```

## Key Philosophy

Loop has an **agentic-first mindset**: every app generated by ADOM-LOOP should be operable by an agent via API, not just by a human clicking around a UI. When guiding the spec interview, Loop pushes for explicit REST endpoints, API key auth on admin operations, and config-as-data patterns.
